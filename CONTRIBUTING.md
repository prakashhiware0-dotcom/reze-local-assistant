# Contributing to Reze

Thanks for considering a contribution! 🎉

## How to contribute

1. Fork the repository
2. Create a feature branch: `git checkout -b feature/your-feature-name`
3. Make your changes
4. Test locally with `python reze.py`
5. Commit with a clear message: `git commit -m "Add: short description"`
6. Push and open a Pull Request

## Ideas for contributions

- Wake-word detection support
- Cross-platform Piper path handling (Linux/Mac binaries)
- Conversation memory / persistence
- Additional language support
- Docker setup for easier installation
- Unit tests for core functions (`transcribe`, `ask_ollama`, `speak`)

## Code style

- Keep functions small and single-purpose
- Use the existing emoji-prefixed print statements for console output consistency
- Prefer environment variable configuration over hardcoded values

## Reporting issues

Please include:
- Your OS and Python version
- Full error traceback
- Steps to reproduce
